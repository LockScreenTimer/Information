# Lock Screen Timer

Lock Screen Timer is an iOS app designed for fast and distraction-free timing directly from the iOS Lock Screen.

The app is perfect for quick workout timers, intervals and rest times, allowing users to start, pause and reset timers without unlocking their phone.

## Features
- Lock Screen widgets for timers
- Three independent timers
- Built-in stopwatch
- Clean and minimal iOS design
